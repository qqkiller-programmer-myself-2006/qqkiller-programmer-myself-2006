## 🤖 AI Agent System

> I build and use **AI-powered coding agent context systems** — every project has an intelligent `.agent/` directory that gives AI assistants full context on the project, rules, logs, and workflows.

```
project/
├── .agent/
│   ├── context.md      ← 🧠 Project brain (AI reads this first)
│   ├── rules.md        ← 📏 Coding standards & git workflow
│   ├── project-log.md  ← 📖 Full history of what was done
│   └── workflows/      ← ⚡ Slash-command workflows
```

This pattern enables **zero-friction onboarding** for any AI assistant — no repeated context, no confusion, just clean output every time.
